# 🚀 BOOSTBACK — 再利用ロケット着陸シミュレータ

ブラウザだけで遊べる、Falcon 9 第一段（ブースター）の着陸ゲームです。
推力・姿勢・風を相手に、脚を展開してパッドへソフトランディングを狙います。
ワンクリックで動く自律着陸モード（Auto Land）も搭載。

▶ プレイはこちら： https://ki-math.github.io/Landing-game/

---

## 特徴

- **3D 表示**：Three.js による A フレーム脚・グリッドフィン・プルーム・着陸パッドの描画と、高度連動カメラ。
- **実機を意識した物理**（操作性のため簡略化）：推力加速度ベース、連続スロットル 0–100%（ホバー ≈ 45%）、姿勢は 2 次系の内ループでトラッキング、風はゆっくり変動する一定風＋突風として横方向に作用。
- **3 段階の難易度**：EASY / NORMAL / HARD（初期高度・降下速度・横オフセット・燃料・風・パッド径・合格判定が変化）。
- **Auto Land（自律着陸）**：ボタンひとつで自動着陸。垂直は高度–速度プロファイル、水平は ZEM 風 PD ＋風フィードフォワードによる誘導。
- **完全オフライン・依存ゼロ**：Three.js を HTML 内に同梱した単一ファイル版あり。ネット接続不要、ダブルクリックで起動。

## 操作方法

| キー | 動作 |
|------|------|
| ↑ ↓ ← → / A D | 姿勢（機体を傾けて移動・離すと復元） |
| W / S | スロットル増減 |
| Space | 全開 |
| X | エンジン停止 |
| G | Auto Land（自律着陸）の ON/OFF |
| C | 視点切替（チェイス / サイド） |
| R | やり直し |

スマホ・タブレットはタッチ操作にも対応しています。

## 着陸の合格条件

接地時に「降下速度・横速度・機体の傾き・パッド中心からの距離」がすべて難易度ごとの許容値以内であれば成功です。脚が無事に展開・接地している必要があります。

## 遊び方（ローカル）

特別な準備は不要です。

- **オフライン版**（`rocket_landing_offline.html`）：ダウンロードしてダブルクリックするだけ。ネット接続なしで動作します。
- **通常版**（`index.html`）：Three.js を CDN から読み込むため、起動時にネット接続が必要です。

## デプロイ（GitHub Pages）

1. 公開したい HTML を `index.html` という名前でリポジトリ直下に置く。
2. リポジトリの **Settings → Pages** を開く。
3. **Build and deployment → Source** を「Deploy from a branch」にし、ブランチ（例：`main`）と `/ (root)` を選択して保存。
4. 数分後、`https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます。
